# Image Segmentation Presentation

Our presentation covers:

- Clustering-based segmentation: Mean Shift
- Graph-based segmentation: Graph Cut
- Probabilistic segmentation: Random Walk
- Recent paper analysis (2024+) connected to graph / clustering / diffusion concepts

## Current Materials

- `segmentation.ipynb`: presentation notebook used as slides in class
- image assets used in the notebook and demos

## How To Run

Use the class Conda environment from the course repository root:

```bash
conda env create -f environment.yml
```

Then open and run the notebook:

```bash
jupyter notebook segmentation.ipynb
```
